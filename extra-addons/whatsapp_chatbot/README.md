# Odoo Chatbot Module for WhatsApp Integration

This module provides a chatbot functionality that can be integrated with the WhatsApp module for automated conversations. It uses the webhook system provided by the WhatsApp API to receive and update messages, and then sets the corresponding response.

## Features

- Integration with WhatsApp API using webhook system.
- Automated response system.
- Allows users to set the communication flow.
- Supports various types of messages such as text, templates, templates with buttons, and Odoo server actions.

## Installation

1. Download or clone this repository into your Odoo addons folder.
2. Restart your Odoo server.
3. Update the app list by going to `Apps -> Update Apps List`.
4. Search for `my_odoo_chatbot_module` and click `Install`.

## Usage

1. Go to the module settings and configure the WhatsApp API integration.
2. Set up the communication flow by defining ordered messages.
3. Choose the type of messages you want to use (text, templates, templates with buttons, Odoo server actions).
4. Save your settings and start using the chatbot.
